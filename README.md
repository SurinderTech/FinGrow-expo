<h1>🎯 About FinGrow</h1>
FinGrow is an **AI-powered financial literacy & fraud awareness** platform developed for the  
**PTU × Punjab & Sind Bank Hackathon 2025**.  
It teaches users smart money management, scam detection, budgeting, and investing.

---

## ✨ Features  
- AI Fraud Simulator  
- Budgeting + Smart Expense Tracking  
- Investment Learning Lab  
- Interactive Quizzes  
- Real-time Alerts & Notifications  
- Multi-language Tutorials  
- Finance Tools: EMI, SIP, Tax calculators  
- Secure Login (JWT + OAuth)

<h1>🚀 Install & Run FinGrow-expo</h1>

```
git clone https://github.com/SurinderTech/FinGrow-expo.git
```

### 1️⃣ Install dependencies
```
cd FinGrow-expo

```
```
npm install
```
```
npx expo start
```

### 2️⃣ Run the app

Scan the QR code with Expo Go on your phone

Or open in Android Emulator / iOS Simulator

Expo Docs → https://docs.expo.dev


Expo Tutorial → https://docs.expo.dev/tutorial
